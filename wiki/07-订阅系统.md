# 订阅系统

## 订阅实体模型

### Subscription 实体

**表名**：`subscriptions`

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| `id` | BIGINT | PK, AUTO_INCREMENT | 主键 |
| `subscriber_id` | BIGINT | FK → users.id | 订阅者（关注者）ID |
| `author_id` | BIGINT | FK → users.id | 被订阅者（作者）ID |
| `created_at` | TIMESTAMP | - | 订阅时间 |

**唯一约束**：`uk_subscription (subscriber_id, author_id)` — 防止重复订阅

**索引**：
- `idx_subscriptions_subscriber (subscriber_id)`：查询我的订阅列表
- `idx_subscriptions_author (author_id)`：查询某作者的粉丝列表

---

## 订阅关系语义

```
用户 A ──关注/订阅──▶ 用户 B（作者）

subscriber_id = A.id
author_id     = B.id
```

- **订阅**：A 关注 B，A 可以在 Feed 中看到 B 发布的内容
- **粉丝**：B 的粉丝包含所有以 B 为 `author_id` 的订阅记录

## API 接口

### 关注用户

```
POST /api/subscriptions/{authorId}
Authorization: Bearer <token>
```

**业务规则**：
- 不能关注自己
- 重复关注返回错误（唯一约束）
- 返回 `SubscriptionDto`（含 subscriber、author 信息及关注时间）

### 取消关注

```
DELETE /api/subscriptions/{authorId}
Authorization: Bearer <token>
```

删除对应的 Subscription 记录。

### 检查是否已关注

```
GET /api/subscriptions/check/{authorId}
Authorization: Bearer <token>
```

返回 `boolean`，用于用户主页的「关注/已关注」按钮状态切换。

### 获取我的订阅列表

```
GET /api/subscriptions/my?page=0&size=20
Authorization: Bearer <token>
```

返回当前用户关注的所有用户列表（`PageResponse<UserDto>`）。

### 获取粉丝列表

```
GET /api/subscriptions/followers/{userId}?page=0&size=20
```

返回指定用户的粉丝列表（无需认证，公开接口）。

### 获取订阅/粉丝数量

```
GET /api/subscriptions/count/{userId}
```

返回：
```json
{
  "subscriptions": 128,
  "followers": 2048
}
```

### 订阅 Feed 流

```
GET /api/subscriptions/feed?page=0&size=10
Authorization: Bearer <token>
```

聚合当前用户所有关注者发布的已发布内容，按 `created_at DESC` 排序，支持分页。

---

## 订阅 Feed 流实现原理

```sql
-- 伪代码逻辑
SELECT c.*
FROM contents c
WHERE c.author_id IN (
    SELECT s.author_id
    FROM subscriptions s
    WHERE s.subscriber_id = :currentUserId
)
AND c.status = 'PUBLISHED'
ORDER BY c.created_at DESC
LIMIT :size OFFSET :page * :size
```

## 前端订阅功能

### 关注/取消关注按钮

在用户主页（`ProfileView`）显示：
- 未关注：「关注」按钮（主色调）
- 已关注：「已关注」按钮（次色调），点击可取消关注
- 自己的主页：不显示关注按钮

页面加载时调用 `GET /api/subscriptions/check/{authorId}` 初始化按钮状态。

### 订阅列表页（SubscriptionsView）

展示当前用户关注的所有用户卡片，可点击跳转到对应用户主页。

### 首页 Feed 切换

`HomeView` 提供 Tab 切换：
- **全部**：调用 `GET /api/contents`
- **订阅**（需登录）：调用 `GET /api/subscriptions/feed`
