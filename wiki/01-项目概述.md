# 项目概述

## 简介

Communication Platform 是一个现代化的在线内容发布平台，支持用户发布文字、图片、视频内容，并提供完整的社交互动功能，包括用户订阅、评论回复、内容搜索以及作者数据管理后台。

## 项目定位

- **类型**：前后端分离的全栈 Web 应用
- **目标用户**：内容创作者与内容消费者
- **核心价值**：提供一个低门槛的内容发布与互动平台

## 核心功能模块

| 模块 | 功能描述 |
|------|---------|
| 用户系统 | 注册、登录、JWT 鉴权、个人资料管理、头像上传 |
| 内容管理 | 发布文字/图片/视频内容、草稿保存、标签分类、CRUD 操作 |
| 评论系统 | 发表评论、嵌套回复（二级）、评论删除 |
| 订阅系统 | 关注/取消关注用户、订阅 Feed 流聚合 |
| 搜索系统 | 按关键词搜索内容与用户、标签搜索、热门标签 |
| 作者后台 | 数据概览（浏览量、订阅数、评论数）、内容管理表格 |
| 媒体上传 | 图片/视频本地存储，支持 JPEG、PNG、GIF、MP4、WebM |

## 系统架构概览

```
┌─────────────────────────────────────────────┐
│                  浏览器 (Vue 3)               │
│  HomeView │ ContentDetail │ Dashboard │ ...  │
└─────────────────────┬───────────────────────┘
                      │ HTTP / REST API
                      ▼
┌─────────────────────────────────────────────┐
│           Spring Boot 3.2 后端               │
│  Controller → Service → Repository → Entity  │
│        Spring Security + JWT Filter          │
└─────────────────────┬───────────────────────┘
                      │ JPA / Hibernate
                      ▼
┌─────────────────────────────────────────────┐
│               MySQL 8.0 数据库               │
│  users │ contents │ comments │ subscriptions │
│                content_tags                  │
└─────────────────────────────────────────────┘
```

## 设计风格

- **主色**：`#FF6B35`（橙红）
- **辅色**：`#F7C59F`（杏色）
- **背景**：`#FFF8F0`（米白）
- **风格**：动态、功能性、暖色调社交应用风格

## 技术栈概览

| 层级 | 技术选型 |
|------|---------|
| 前端框架 | Vue 3 + TypeScript + Vite |
| UI 组件库 | Element Plus（暖色调定制） |
| 状态管理 | Pinia |
| 路由 | Vue Router 4 |
| 后端框架 | Spring Boot 3.2 |
| 认证 | Spring Security + JWT |
| ORM | Spring Data JPA / Hibernate |
| 数据库 | MySQL 8.0 |
| 数据库迁移 | Flyway |
| 测试 | JUnit 5 / Vitest / Playwright |
| 容器化 | Docker + Docker Compose |
| 包管理（前端）| pnpm |

